Задание на лабораторную работу 1

Из списка A удалить те цепочки нечетных элементов, в которых нет ни одного элемента из списка B.
Пример: список A[10]: 3 2 7 5 2 1 2 6 3 9
список B[5]: 1 2 5 4 8
список A после удаления примет вид:
A[7]: 2 7 5 2 1 2 6.

Лабораторная работа №1. Обработка списков
1.	Ввод элементов списка должен быть доступен с клавиатуры и путем автоматической генерации
2.	Программа должна корректно обрабатывать некорректный ввод.
3.	Исходный код должен быть откомментирован
4.	Необходимо реализовать правильную декомпозицию программы на методы.
5.	Необходимо реализовать предложенный алгоритм без использования стандартных функций и с использованием (необходимо реализовать два отдельных метода).
