Лабораторная работа №2
Выполнить обработку элементов прямоугольной матрицы A, имеющей N строк и M столбцов. Определить, сколько отрицательных элементов содер- жится в каждом столбце и в каждой строке матрицы. Результат  оформить   в виде матрицы из N + 1 строк и M + 1 столбцов.

1.	Ввод элементов списка должен быть доступен путем автоматической генерации. Необходимо использовать библиотеку numpy. Результаты выполнения должны сохраняться в файл (исходные данные и результат обработки).
2.	Исходный код должен быть откомментирован
3.	Необходимо реализовать правильную декомпозицию программы на методы.
